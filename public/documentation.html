Movie API

Welcome to the Movie API documentation. This API provides information about various movies, their directors, and genres. Users can sign up, update their information, and create a list of their favorite movies.

Request URL Response
GET /movies Returns a list of all movies.
GET /movies/:title Returns data about a single movie by title (description, genre, director, image URL, featured status).
GET /genres/:name Returns genre description by name.
GET /directors/:name Returns director’s bio, birth year, and death year by name.
POST /users Registers a new user.
PUT /users/:username Updates user information (e.g., username).
POST /users/:username/movies/:movieID Adds a movie to the user’s list of favorites. Returns confirmation text.
DELETE /users/:username/movies/:movieID Removes a movie from the user’s list of favorites. Returns confirmation text.
DELETE /users/:username Deregisters an existing user. Returns confirmation text.
